# LinkBot-AI 全域获客智能客服系统 - 项目总结

## 🎯 项目概述

**LinkBot-AI** 是一个基于 0-代码理念设计的全域获客智能客服系统，支持抖音、快手、视频号、小红书等主流直播平台，通过 AI 智能回复和线索管理，实现公域流量到私域线索的自动转化。

## 🏗️ 技术架构

### 前端 (React + TypeScript + Ant Design)
- **位置**: `/frontend`
- **技术栈**: React 18, TypeScript, Ant Design, React Router, Recharts
- **功能**: 白标管理后台，支持多租户，实时仪表盘，渠道管理，对话管理，线索中心
- **部署**: Railway Static 服务

### 后端 API (Node.js + Express + TypeScript)
- **位置**: `/backend`
- **技术栈**: Node.js, Express, TypeScript, JWT, bcryptjs
- **功能**: RESTful API，用户认证，数据管理，AI 集成
- **部署**: Railway 容器服务

### 渠道代理 (Go + WebSocket)
- **位置**: `/proxy`
- **技术栈**: Go 1.21, WebSocket, 限流器
- **功能**: 多平台 WebSocket 连接，事件解码，消息转发
- **部署**: Railway 容器服务

### 数据库 (PostgreSQL + Redis)
- **PostgreSQL**: 主数据库，存储用户、对话、线索等数据
- **Redis**: 缓存和会话管理
- **部署**: Railway 托管服务

## 🔧 部署方案

### 本地开发
```bash
# 启动所有服务
docker-compose up -d

# 或分别启动
cd frontend && npm start
cd backend && npm run dev
cd proxy && go run main.go
```

### 生产部署 (Railway)
```bash
# 一键部署
./deploy.sh

# 或手动部署
railway login
railway init
railway add postgresql
railway add redis
railway up
```
