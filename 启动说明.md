# CEO，LinkBot-AI 启动说明

## 🚀 一键启动

### 启动所有服务
```bash
cd /Users/yiche/linkbot-ai
bash start-all.sh
```

### 停止所有服务
```bash
cd /Users/yiche/linkbot-ai
bash stop-all.sh
```

## 📋 服务清单

启动脚本会自动启动以下服务：

1. **后端服务** - 端口 3001
   - 提供 API 接口
   - 日志：`tail -f /tmp/backend.log`

2. **Go代理服务** - 端口 8080
   - 处理抖音OAuth和WebSocket
   - 日志：`tail -f /tmp/proxy.log`

3. **前端服务** - 端口 3000
   - React + Ant Design UI
   - 日志：`tail -f /tmp/frontend.log`

## 🎯 访问地址

- **前端登录页**：http://localhost:3000/login
- **登录账号**：admin / admin123
- **后端API**：http://localhost:3001
- **Go代理**：http://localhost:8080

## ⚠️ 注意事项

1. 首次启动需要等待前端编译（约1-2分钟）
2. 所有服务启动后会自动检查状态
3. 停止服务请使用 `stop-all.sh` 脚本
